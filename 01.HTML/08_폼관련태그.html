텍스트와 관련된 input 태그들

▶ type="text"

- 한 줄 짜리 텍스트를 입력할 수 있는 텍스트 상자이다.

type="text"에서 새로 추가된 기능

- datalist 태그를 사용하여 텍스트 필드에서 데이터 목록 중에서 값을 선택하도록 만들 수 있다.

▶ type="password"

- 비밀번호를 입력할 수 있는 텍스트 상자이다.

▶ type="search", type="url", type="email", type="tel"

- 겉 모습은 텍스트 필드와 비슷하지만 각각의 정보에 맞게 세분화된 기능을 제공하는 텍스트 상자이다.



숫자와 관련된 input 태그들

▶ type="number"

- 입력창에 숫자를 입력하는 것이 기본이지만 브라우저에 따라 스핀 박스가 표시되기도 한다. (스핀박스 : 입력창 오른쪽에 위/아래 화살표 버튼)

▶ type="range"

- 슬라이드 바를 통한 숫자 지정 가능

날짜/시간 관련 태그

▶ type=""date|month|week|time|datetime|datetime-local"











라디오 버튼과 체크박스 태그

▶ type="radio"

- 태그 여러개를 선언하여 하나의 용도로 사용
- 라디오 버튼의 경우 관련된 버튼끼리는 name속성이 같아야 하나만 선택이 가능하다.



▶ type="checkbox"

- 여러 항목 중 여러가지 항목을 선택해야 할 경우 사용하는 태그이다.
- 체크 박스의 경우 관련된 체크박스 기리는 name속성이 같아야 한다.
- value속성은 선택한 라디오 버튼이나 체크박스의 서버로 넘길 값을 지정한다.
- checked 속성은 기본적으로 선택할 항목이 있다면 사용된다.
